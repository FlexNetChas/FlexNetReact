# Flexnet

Flexnet is an innovative web application designed to help users find the right higher education courses based on their academic performance. It also provides recommendations for the grades users need to improve in order to meet admission requirements. Whether you're aiming for your dream job or just exploring educational options, Flexnet is your personalized guide to the right studies.

---

## Features

* **Personalized Recommendations**: Based on your current grades, Flexnet helps you discover higher education opportunities that match your qualifications.
* **Grade Improvement Insights**: Provides tailored suggestions on which grades you need to improve to meet program entry requirements.
* **Interactive 3D Features**: Using Three.js, Flexnet includes immersive 3D elements to enhance the user experience.
* **Future Enhancements**:

  * **Voice Command Integration**: Enabling voice control for hands-free navigation.
  * **Grade Management & Analytics**: Track your grades and academic progress, and plan your path toward your dream career.
  * **Personalized Avatars**: Customize your profile with unique avatars and styling options.

## Tech Stack

* **React**: Efficient, component-based UI library.

## Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js (v14.x or later)
* npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/flexnet.git
   ```

2. Install dependencies:

   ```bash
   cd flexnet
   npm install
   # or
   yarn install
   ```

3. Set up environment variables:

   * Create a `.env.local` file in the root of the project.
   * Add required API keys, database URLs, etc.

4. Run the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

   This will start the app on [http://localhost:3000](http://localhost:3000).
